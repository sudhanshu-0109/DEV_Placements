# 🎯 Week Objective

This week is all about building a **strong foundation in React**.

By the end of Week 1, you should:

- Understand how React works internally
- Create React applications using Vite
- Build reusable components
- Pass data using Props
- Handle user interactions
- Manage state using `useState`
- Complete your first React mini-project

> **Goal:** Stop thinking in HTML pages and start thinking in **Components**.

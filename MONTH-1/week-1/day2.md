# 📝 Today's Notes

## Components

```
A component is an independent, reusable piece of UI.
Large applications are built by combining small components together.
```

---

## Props

```
Props allow a parent component to pass data to a child component.
They make components reusable and dynamic.
Props are read-only.
```

---

# 🏗 Engineering Practice

Today I learned to think in components instead of pages.

Instead of asking:

> "How do I build this page?"

I should ask:

> "What reusable components make up this page?"

Examples:

- Navbar
- Sidebar
- Card
- Button
- Footer
- Modal
- Form

Each should become an independent component.
